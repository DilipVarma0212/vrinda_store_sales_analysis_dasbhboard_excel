# Vrinda Store Sales Analysis Dashboard

This project focuses on analyzing sales data of Vrinda Store, an online clothing retailer. The dataset contains order-level information for the year 2022.


## Project Overview

The main objective of this project is to create an interactive Excel dashboard and generate insights that help improve business performance in 2023.


## Objectives

- Analyze yearly sales performance
- Identify customer behavior and buying patterns
- Find top-performing states and sales channels
- Understand product and demographic trends
- Provide actionable business insights


## Data Source

Sales Data: The primary dataset used for this analysis is the "vrinda-store-data-analysis.xlsx" file, containing detailed information about each sale made by the store.


## Tools

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning Techniques
- Excel Functions (SUM, COUNT, VLOOKUP)


## Data Cleaning Process

1. Data loading and inspection.
2. Removed duplicate records.
3. Handling missing values.
4. Data cleaning and formatting.
5. Standardized data formats (dates, text).


## Dashboard Features

- Monthly Sales vs Orders Comparison
- Sales by Gender
- Sales by Age Group
- Top 5 States by Sales
- Sales by Channel
- Order Status Distribution


## Results

- Women contribute approximately 65% of total purchases.
- Maharashtra is the top-performing state, followed by Karnataka and Uttar Pradesh.
- Age group 30–49 years contributes around 50% of total sales.
- Amazon is the leading sales channel, followed by Flipkart and Myntra.


## Recommendations

- Focus marketing efforts on women customers.
- Target 30–49 age group for higher conversion.
- Expand operations in top-performing states.
- Prioritize Amazon channel for maximizing revenue.
- Improve strategies for low-performing regions.


## Project Outcome

- Developed an interactive dashboard for easy data analysis.
- Generated meaningful insights for decision-making.
- Helped identify key growth opportunities for 2023.
